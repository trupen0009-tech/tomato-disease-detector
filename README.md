# Tomato Disease Detector

AI model that detects diseases in tomato leaves.

## Diseases Detected
- Late Blight
- Early Blight
- Septoria Leaf Spot

## Live Demo
https://huggingface.co/spaces/Trupen007/plant-leaf-detector

## How to Use
1. Open the live demo link
2. Model predicts the disease instantly

## Dataset
Sample test images available on request. Model trained on public tomato leaf disease dataset.

## Built With
Python, Machine Learning, Gradio
